- 500 مقطعاً مُولَّداً. **تصحيحات** (تُعدِّل النطق فقط، المفتاح يبقى أصلياً):
  - `SPEAK_FIX`: «تَ» كانت تُنطَق «ت فتحة» → «تَـ»؛ «يِ» كانت تُمَدّ → `[[j'i]]`؛ «آ» → `[[?'a:]]`؛ «أً» → `[[?'an]]`؛ كلمات العين.
  - `weakFix`: العين (A) والغين (Q) ضعيفتان → إطالة فونيمية `A:`/`Q:` في كل أشكالهما؛ وسكونهما نقيٌّ ممتدّ `[[A:]]`/`[[Q:]]`.
  - **السكون** = صوتُ الحرف مجرّدًا دون أيّ إضافة (قرار المستخدم).
- **حدّ espeak مُثبَت بالقياس (−91dB صمت):** السكون على الانفجاريّة المجهورة **ب ج د ض** + الهمزة **أْ** = صمتٌ تامّ. لا يُحَلّ دون حركة (مرفوضة). بقيّة الـ23 سليمة.

### طبقة التسجيل البشريّ (تجاوز espeak) — `src/tts-clips.js`
- تسجيلٌ بشريّ في `public/tts/custom/<hash>.mp3` يُقدَّم على مقطع espeak. `src/tts-custom.json` قائمة الموجود (يبنيها `npm run gen:custom`). `public/tts/custom/README.md` فيه أسماء ملفّات الحروف الخمسة المطلوبة.

### نظام نماذج الصوت الكامل (وضع التهجئة + الاستوديو + المجموعات)
- **وضع التهجئة** (`src/spell.js`): «🐢 هجِّ ببطء» في `read.html` — تركيب الكلمة من مقاطعها ثم كاملةً.
- **اختيار نموذج الصوت لكل نوع** (حرف/كلمات/جمل) في لوحة الأهل: «مقاطع مُولَّدة (espeak)» أو مجموعةٌ بشريّةٌ بالاسم (مُدمجة أو على الجهاز). `sound-prefs.getVoiceSet/setVoiceSet`، `tts-clips.classify+srcFor`.
- **استوديو التسجيل** (`record.html`): 3 تبويبات/وحدات + ملفّ مُسجِّل (اسم/جنس/فئة) + تسجيل IndexedDB لكل مجموعة (`src/voices.js`) + ضمّ المجموعات + تصدير ZIP مصنّف (`src/zip.js`).
- **الدمج مع الحزمة**: `npm run gen:voices` (من `voices-src/` المُصدَّر) → `public/tts/voices/` + `src/voices-bundled.json`.
- **مصدر مفردات موحّد** (`src/vocab.js` + `src/robo-phrases.js`): الاستوديو والمولّد يقرآن منه → تزامنٌ تلقائيّ مع المحتوى.
- **أوّل صوتٍ بشريّ مُدمج**: «صالح» (632 مقطعاً: حروف+كلمات+جمل+عبارات).
- **نتيجة تجربة Piper/MMS**: العصبيّ غير مناسبٍ للحروف المنفردة والشكل؛ صالحٌ للكلمات/الجمل فقط. القرار: espeak افتراض + بشريّ للجوهر.
