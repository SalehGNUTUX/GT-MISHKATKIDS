# تنسيق المحتوى ونموذج الشفاء — «تلميذي»

> كيف نؤلّف «ذكريات» روبو كبيانات محليّة (بلا ذكاء اصطناعي). التطبيق يقرأ هذا التنسيق ويُرتّبه عبر آلة الحالات.
> البيانات الفعليّة في: [`content/library.js`](../content/library.js).
> آخر تحديث: 2026-06-10

---

## 🧠 نموذج شفاء روبو
كل ذكرى يستعيدها الطفل تُضيء جزءًا من روبو أو تُنبت عنصرًا في عالمه.

**أجزاء روبو (`robo_part`):** `العينان` (البصر/الملاحظة) · `الأذنان` (الإصغاء) · `الفم` (الكلام) · `القلب` (المشاعر) · `اليدان` (الصنع) · `الساقان` (الحركة) · `الدماغ` (التفكير/الذاكرة).

**عناصر العالم (`world`):** `قمر` · `شمس` · `شجرة` · `صديق` · `نهر` · `نجمة` · `طائر`.

**الضحكة 🎉 (الجائزة الكبرى):** ليست ذكرى، بل **معلَم** يتحقّق حين يُستعاد عددٌ كافٍ (مثلًا ≥ 8 مكافآت) — تقرّره آلة الحالات، فتعود ضحكة روبو ولونه الكامل.

**لوحة الألوان العائدة:** دافئة تدريجيًّا — `#F4C95D` `#F6BE3B` `#E0566B` `#7BB661` `#6FB3D6` `#C9A0DC` `#E08E45`.

## 🧩 مخطّط الذكرى (Memory)
| الحقل | النوع | المعنى |
|---|---|---|
| `id` | نص (kebab) | معرّف ثابت فريد |
| `title` | نص | عنوان قصير (للوحة الأهل) |
| `domain` | تعداد | `أرقام وهندسة` · `حروف ولغة` · `أخلاق ومشاعر` · `علوم وطبيعة` · `تفكير وحلّ مشكلات` |
| `concept` | نص | ما الذي يستعيده روبو |
| `age_band` | تعداد | `4-6` أو `7-10` |
| `duration_min` | رقم | تقدير زمن المهمة (للجلسة الواحدة) |
| `multi_day` | منطقي | مهمّة تمتدّ أيّامًا؟ |
| `mode` | تعداد (اختياريّ) | `solo` (افتراضيّ عند الغياب) أو `pair` — ذكرى للإخوة معًا |
| `inspired_by` | نص (اختياريّ) | مصدر الإلهام (مثلًا `GT-SARARIM`) — انظر [CREDITS.md](CREDITS.md) |
| `spark` | كائن | `{ robo_line, cue, audio? }` — شرارة الذكرى |
| `task` | كائن | `{ prompt, child_steps[], materials[], parent_role, location }` |
| `teach` | كائن | `{ success_signal, robo_question, capture[] }` |
| `heal` | كائن | `{ robo_reaction, reward:{ type, target, color }, audio? }` |
| `tags` | قائمة نصوص | للبحث والتصفية |

- `capture[]` ∈ `tap_done` (الطفل/الأب يضغط «تمّ») · `voice` (تسجيل صوتيّ محليّ) · `photo_local` (صورة تبقى على الجهاز).
- `reward.type` ∈ `robo_part` أو `world`؛ و`target` من القائمتين أعلاه؛ و`color` لونٌ من اللوحة.
- `audio` اختياريّ: مسار لتسجيلٍ مسبق لصوت روبو (أدفأ من التركيب الآليّ).
- `mode: "pair"` يميّز **ذكريات الإخوة** (تعاونيّ — طفلان يُعلّمان روبو معًا)، ويُظهرها التطبيق والمطبوعات بشارة «👯 لطفلين معًا».

## ✍️ إرشادات التأليف
- **جملة روبو قصيرة ودافئة** بصيغة «نسيتُ… ساعِدني» — يبقى الطفل هو المعلّم الواثق، وروبو هو من يُخطئ.
- **المهمّة واقعيّة يدويّة** بأشياء البيت، 15–30 دقيقة (أو متعدّدة الأيّام).
- **4–6:** حسّية بسيطة، الأب/الأم رفيقٌ طيّار، الإثبات باللمس/الصورة.
- **7–10:** سؤال «لماذا» أعمق، خطوات أكثر، والطفل يشرح بكلماته أو يسجّل صوته.
- **`success_signal` واضحٌ** ليقرّره الأهل بأنفسهم (لا ذكاء يحكم على الطفل).
- **خصوصية:** الصور والتسجيلات تبقى على الجهاز فقط.
