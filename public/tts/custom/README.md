# تسجيلات النطق البشرية (تجاوز espeak)

محرّك espeak يَعجز عن نطق **السكون** على الحروف الشديدة المجهورة (صمتٌ تامّ): **ب، ج، د، ض**، وكذلك **الهمزة الساكنة (أْ)**. الحلّ: ضع هنا تسجيلاً بشريًّا قصيراً للحرف الساكن، فيُشغَّل بدل مقطع espeak تلقائياً.

## الحروف التي تحتاج تسجيلاً (سكون):

| الحرف الساكن | اسم الملفّ المطلوب |
|---|---|
| بْ | `3099363eedc38149.mp3` |
| جْ | `3bb6c43eb92da707.mp3` |
| دْ | `6b954c3951328d90.mp3` |
| ضْ | `c3d510680e2d9c02.mp3` |
| أْ | `bd94f4eafbf82b43.mp3` |

## الخطوات:
1. سجّل الحرفَ ساكناً (مثلاً «بْ» بقلقلةٍ خفيفة دون حركة) كملفّ MP3.
2. سمِّ الملفّ بالاسم المقابل في الجدول أعلاه، وضعه في هذا المجلّد `public/tts/custom/`.
3. شغّل: `npm run gen:custom` ثم `npm run build`.

> لأيّ مقطعٍ آخر تريد استبداله بصوتٍ بشريّ: ابحث عن نصّه في `src/tts-manifest.json` لتعرف اسم ملفّه، ثم ضع تسجيلك بالاسم نفسه هنا.
