## أوامرُ التطوير الأساسيّة (من داخل `GT-MK/`)

```bash
npm run dev          # خادمُ تطوير Vite (المنفذ 5174 مثبَّت)
npm run build        # بناءُ الإنتاج → dist/ (إلزاميٌّ بعد أيّ تعديل)
npm run gen:audio    # مقاطعُ espeak (يتطلّب espeak-ng + lame)
npm run gen:piper    # مقاطعُ Piper العربيّة العصبيّة (ar_JO-kareem)
npm run gen:icons    # توليدُ أيقونات SVG من Font Awesome → src/icons.js
node tools/gen-piper-lang.mjs en|fr   # أصواتُ اللغات الأجنبيّة
npm run gen:quran    # نصُّ القرآن + تلاوة الحصري (إنترنت مرّةً، وقتَ التطوير)
```

## مبادئُ ثابتةٌ (لا تُكسَر)
- **محلّيٌّ 100%** دون إنترنت ودون ذكاءٍ وقتَ التشغيل.
- الآليُّ لا يُوصَف بـ«حيّ». نصوصُ المستودع كلُّها **عربيّةٌ فصحى**.
- لا يُحذَف محتوًى/ميزةٌ دون إذن (نُثري لا نحذف). الأرقامُ غربيّةٌ في الواجهة.
- **لا رفعَ حتى الاختبار المحلّيّ** وموافقةُ المستخدم.
